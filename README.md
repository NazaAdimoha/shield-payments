- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Animations**: Framer Motion
- **Data Fetching**: React Query
- **Tables**: TanStack Table
- **Theme**: next-themes
